 1. Создать внешний репозиторий c названием TXT:
 GitHub > Repositories -> New -> TXT

 2. Клонировать репозиторий TXT на локальный компьютер:
  git clone github.com/elvyudina/TXT.git

 3. Внутри локального TXT создать файл “new.txt”:
 cd TXT
 touch new.txt

 4. Добавить файл под гит:
 add new.txt

 5. Закоммитить файл:
  git commit -m "new TXT file"
 6. Отправить файл на внешний GitHub репозиторий:
 git push

 7. Отредактировать содержание файла “new.txt” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). Всё написать в формате TXT:
 vim new.txt
 Нажимаем i,для редактирования файла
 Фамилия имя отчество: Юдина Елена Викторовна.
 Возраст:37.
 Количество домашних животных:1.
 Будущая желаемая зарплата:30000.
 нажимаем Esc, :wq

 8. Отправить изменения на внешний репозиторий:
 git commit -am "added information about yourself"
 git status
 git push

 9. Создать файл preferences.txt
 touch preferences.txt 

 10. В файл preferences.txt” добавить информацию о своих предпочтениях (Любимый фильм, любимый сериал, любимая еда, любимое время года, сторона которую хотели бы посетить) в формате TXT:
 vim preferences.txt
 Нажимаем i,для редактирования файла
 Любимый фильм: Мистер и миссис Смит.
 Любимый сериал: Теория большого взрыва.
 Любимая еда: мясо по французски.
 Любимое время года: лето.
 Страна которую хочу посетить: Италия.
 нажимаем Esc, :wq 

 11. Создать файл skils.txt добавить информацию о скиллах которые будут изучены на курсе в формате TXT
touch skills.txt
 vim skills.txt

                                структуры HTTP запросов и ответов,

 нажимаем Esc, :wq
